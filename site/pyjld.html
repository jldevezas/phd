PyJLD: A workbench for latent factor models

As part of the PhD work, we've developed an open source Python library to experiment with latent factor models. We've implemented SVD-based rating prediction and recommendation, and a k-fold cross-validation method for the evaluation of the model and to compare the SVD baseline to any further improvements we plan to add based on the social dimension. Additionally, we've also extended the cross-validation method to enable the identification of the number of latent factors that minimizes the mean absolute error.

Download jld.py from GitHub: git@github.com:jldevezas/phd.git

You can also get an R ggplot2 script from here to make a cool-looking chart like the one on the right. It shows the minimum mean absolute error averaged over k=10 folds and the number of latent factors that produce that value. The script can directly be used with the cross_validation.py CSV generated with --output when using the --feature-sampling-interval argument

Manual

PyJLD consists of the library jld.py and the following command line tools that implement and exemplify the features in this library: train.py, predict.py, recommend.py, recommend_by_query.py, nn.py, cross_validation.py and server.py.

jld.py

This library will comprise all the Python classes that I develop during my doctoral project. For now, however, it only contains a single class named LatentFactorsModel that implements rating prediction and recommendation using Singular Value Decomposition (SVD), as well as k-fold cross-validation to evaluate your model and compare to any future implementations.
