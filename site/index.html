Listening Behavior

We present an exploratory analysis, supported on data visualization, depicting the hourly, weekly and monthly behavior, globally as well as for individual users, based on the Last.fm dataset provided by Oscar Celma, which comprises scrobble information for nearly 1,000 users crawled randomly from his neighborhood. This data can be found here.

Available analyses include a set of user-centric interactive visualizations, as well as a set of global overview plots.
