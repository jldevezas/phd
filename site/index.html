Theory

I take quite a pride in understanding concepts with enough detail, so that I can then convey the ideas in layman's terms. I believe that the mastery of a subject requires being able to transmit knowledge in a way that makes it seem trivial for the learner.

In the theory sections, I present this kind of approach, trying to pass knowledge as if I were explaining it to a friend, instead of formally as is the case of the majority of the bibliography out there. This is the kind of contribution you will find in this website. If however you're still interested in the formal approach of this same content, you can read my thesis when I finish my PhD, around the year 2017. For now, you can read about singular value decomposition and k-fold cross-validation in the context of recommender systems.

Tools

In the process of understanding the algorithms I'm studying and as an engineer, I'm implementing code that will function as my workbench, but also as an open source resource for anyone interested in the same subjects. Even if not quite production-ready, it poses a good resource to easily test and extend on recommendation algorithms.

Most of these implementations will be available through PyJLD, a Python library for the study of recommender systems. The mission is to implement and experiment with several recommendation algorithms, but also to integrate into the recommendation process any knowledge obtained through algorithms that harness the information of social networks, including the community structure. Therefore, we also make it the mission of PyJLD to provide implementations to algorithms for community detection and social network analysis in general.
